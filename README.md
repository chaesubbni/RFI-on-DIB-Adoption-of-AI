# RFI-on-DIB-Adoption-of-AI

### 논문 목적
- 미 국방부가 방산 산업이 AI를 도입하는 데 어떤 장애요인이 있는지 파악하기 위해 설문을 진행했고 그 응답을 분석.

### 응답자 구성
- 대기업
- 중소기업
- 협회
- 컨설팅
- 개인

### 방산 AI 도입의 4대 장애요인
- 인프라, 공급망 문제
    - AI용 반도체, GPU, 데이터센터 등 핵심 인프라가 해외 의존도가 높음.
    - 특히, 중국 의존도가 위험 요소로 지적됨.
    - 생산 속도가 AI 수요를 못 따라감.
    - 공급망 공격(IP 도난, 사이버 공격) 위험 증가
- 인력 부족
    - AI 전문 인력 부족
    - 교육 부족
- 혁신 장애요인
    - 데이터가 부처별로 분산되어 있어 접근 어려움.
    - 보안 때문에 데이터 공유가 거의 안 됨
    - IP 문제 때문에 민간 기업이 기술 공유를 꺼림.
- 획득, 정책, 규제 문제
    - 방산 조달 절차가 너무 느리고 복잡함.
    - 중소기업이 참여하기 어려움
    - AI 특성에 맞지 않는 규제


### 결론
- 방산 AI 도입은 기술 문제가 아니라 구조적, 제도적 문제가 더 크다.
- 공급망, 데이터, 인력, 조달 시스템이 개선되어야 AI가 본격적으로 확산될 수 있다.
